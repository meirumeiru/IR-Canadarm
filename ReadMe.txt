IR-Canadarm
A mod for Kerbal Space Program 1.12.x that provides Canadarm and Canadarm2 parts based on Infernal Robotics Next.

The mod is derived from HABTECH ROBOTICS which is a mod for Kerbal Space Program [Breaking Ground Expansion] 1.12.x by Benjee10.
